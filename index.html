Nos outils SEO

Découvrez nos outils pour optimiser votre site web et améliorer votre visibilité en ligne.

Générateur de balises méta

Notre Générateur de Balise Meta est l'outil idéal pour optimiser votre site web pour les moteurs de recherche. Créez rapidement et facilement des balises meta personnalisées, y compris les balises title, description et mots-clés, pour chaque page de votre site. Améliorez la visibilité de votre site sur les moteurs de recherche et attirez plus de trafic qualifié grâce à des balises meta bien structurées et pertinentes.

Acceder à l'outil

Testeur de robots.txt

Testez et validez votre fichier robots.txt. Vérifiez si une URL est bloquée et comment. Vous pouvez également vérifier si les ressources de la page sont interdites.

Testeur .htaccess

Testez vos règles de réécriture .htaccess pour les redirections HTTP (côté serveur). Cet outil s'appuie sur l'API développée par l'équipe madewithlove.

Générateur de sitemap

Générez rapidement et facilement un plan de site XML avec des balises alternatives (par exemple des balises hreflang) à partir d'un document Excel (.csv). Vous pouvez également « pinger » les moteurs de recherche (Google et Bing) pour obtenir des plans de site XML mis à jour à l'aide d'une simple requête HTTP.

Analyseur de flux RSS

Extrayez rapidement les URL et les dates de publication des flux RSS.

Testeur de balises hreflang

Les annotations hreflang peuvent être difficiles à implémenter correctement. Cet outil vous permet de vérifier rapidement si les balises hreflang d'une page (en-têtes HTML et HTTP) ou dans les plans de site XML sont correctes.

Simulateur SERP

Cet outil fournit un aperçu d'extraits Google pour vérifier comment les balises de titre et les méta-descriptions sont affichées dans les résultats de recherche. Google peut les modifier, mais les éléments méta bien optimisés sont plus susceptibles d'apparaître dans les SERP.

Générateur de schémas

Un générateur de données structurées Schema.org qui prend en charge la création de balises JSON-LD. Y compris toutes les propriétés d'élément requises et plus encore.

Suivi des mises à jour d'Algo

Suivez toutes les mises à jour confirmées et non confirmées de l'algorithme Google. La liste est actualisée régulièrement.

Analyser les liens d'une page

Cet outil permet d'analyser les liens présent sur n'importe quel page web. Cela permet de savoir combien il y en a, combien son en nofollow (ou pas), si un lien vers une même page est présent plusieurs fois et si les liens sont des interne ou externe au site. Il y a aussi le PageRank de cette même page qui est calculé. Cela permet ainsi de savoir si la page peut retransmettre facilement son PageRank ou non. Une page avec un petit PageRank peut ainsi donner plus de poids qu'une page avec un fort PageRank mais beaucoup de liens sortant.

Analyse de balises meta

Cet outil permet d'analyser vos balises meta sur l'une de vos page web. Vous avez juste à indiquer l'adresse de la page à analyser et cet outil va lire vos balises meta pour ensuite faire une synthèse.

Analyse Taux Texte/Code

Cet outil analyse le code source d'une page web et vérifie la densité du contenu textuel comparé au code HTML de la page. Un ratio est donné pour mieux se rendre compte s'il y a trop de balises HTML
